# Support

Thanks for using GentleDuck.

If you need help:

- Check docs: https://gentleduck.org
- Join discussions: https://github.com/gentleeduck/duck-ui/discussions
- Report bugs: https://github.com/gentleeduck/duck-ui/issues/new?template=bug_report.md
- Request features: https://github.com/gentleeduck/duck-ui/issues/new?template=feature_request.md

Please do not use GitHub Issues for general support requests.
