## Status

`@gentleduck/primitives/layer` is currently a reserved entrypoint.

`packages/duck-primitives/src/layer/index.ts` does not expose stable public exports right now.

---

## Current implementation state

- The layer entrypoint is intentionally kept as a placeholder.
- Related behavior lives in active primitives such as dismissable layer and portal modules.

---

## Related active primitives

Related active primitives:

- `@gentleduck/primitives/dismissable-layer`
- `@gentleduck/primitives/portal`

---

## Migration note

If you previously depended on internal layer helpers, move to public APIs under `dismissable-layer`, `portal`, and `focus-scope` until a stable `layer` export is introduced.
